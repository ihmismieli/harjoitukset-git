# Git-versionhallinta Hello-ohjelma

Tässä repositoriossa on kurssin **Git-versionhallinta - SOF013AS2A-3001** harjoitukset Hello-ohjelmaan liittyen.

Tehtävissä harjoiteltiin versionhallinnan keskeisiä periaatteita: haarojen luomista ja yhdistämistä, commitien tekemistä, muutosten seuraamista sekä etärepositorion käyttöä GitHubissa.  
Harjoitusten avulla opittiin myös, miten versionhallintaa hyödynnetään sekä yksin työskennellessä että tiimiprojekteissa.
